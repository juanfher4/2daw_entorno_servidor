Bienvenido al programa de Gestión de Base de Datos

Introducción

A través de esta Web se van a realizar a modo didáctico una serie de operaciones sobre bases de datos usando PHP.

Para seguir la operación deseada debe pulsar sobre alguno de los enlaces del menú izquierdo o superior.

BREVE AYUDA SOBRE CADA UNO DE LOS ENLACES DEL MENÚ

Consultar: Permite consultar todos los registros almacenados en las tablas de proveedores proveedores(S), piezas(P) y proveedores-Piezas(SP).
Insertar: Permite consultar e insertar registros almacenados en las tablas de proveedores proveedores(S), piezas(P) y proveedores-Piezas(SP).
Modificar: Permite modificar registros almacenados en la tabla de proveedores(S).
Borrar Registros: Permite borrar registros almacenados en la tabla de proveedores(S), piezas(P) y proveedores-piezas(SP).
Borrar Tabla: Permite borrar toda una tabla completa de la base de datos. Los posibles tablas a borrar son las de proveedores proveedores(S), piezas(P) y proveedores-Piezas(SP).
Ejecutar SQL: Permite ejecutar cualquier sentencia SQL sobre cualquier tabla de cualquier base de datos.
Consultas Predefinidas: Permite ejecutar una serie de consultas establecidas. Se mostrará el resultado de dicha ejecución de consulta por pantalla.
